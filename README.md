# Set fan speed

## Command

```
SET_FAN_SPEED:<SPEED>
```

## Arguments

| Command name | Type | Range   |
| ------------ | ---- | ------- |
| SPEED        | int  | 0%-100% |

## Examples

### Set the fan speed at 50%

```
SET_FAN_SPEED:50
```
